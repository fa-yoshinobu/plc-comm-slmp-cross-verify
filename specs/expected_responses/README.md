# Expected Responses

This directory contains the checked-in response expectations that are not
derived mechanically from a fresh mock-baseline run.

Current scope:

- `live_profiles.json`
  Target-specific live-replay overrides keyed by verification case name.

## What Stays Here

- validated PLC profile overrides such as `r120pcpu_tcp1025`
- case-specific live comparison mode changes
- target-specific expected response bytes, end codes, or payload lengths

## What Does Not Stay Here

- mock baseline response packets for every case

Those are generated by `python verify.py` and written to
`logs/latest_live_cases.jsonl`. `slmp_live_verify.py` replays against that file
and then applies any matching overrides from `live_profiles.json` that were
embedded by `verify.py`.
